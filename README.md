# Dimensions
borwell software challenge / sample Unity application
by Jon Rummery (jonrummery@gmail.com)

Build instructions :
(1) clone repository
(2) open with Unty (2018.2.1f1 or later). 'File' -> 'Open Project'
(3) play. 'File' -> 'Play'
